# JAGD Construction Cert Portal (Launchable Demo)

This is a launchable Node/Express demo that serves a working front end and API from one server.

## Test logins
- admin / admin123
- office / office123
- pm / pm123

## Run
1. Open a terminal in this folder
2. Run:
   npm install
   npm start
3. Open:
   http://localhost:3001

## What is included
- Login screen
- Dashboard
- Employees
- Worker profile
- Jobs
- Add Job
- Edit Job requirements
- Bloodwork
- Upload queue
- Reports
- Admin
- Real imported worker names from the 2026 workbook
- 30 current jobs preloaded

## Notes
- Data is stored in `data/store.json`
- This is a demo-ready launchable build, not a production-hardened system
- Real auth, secure uploads, backups, and deployment hardening should be added before public launch
